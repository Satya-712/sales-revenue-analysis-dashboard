# Sales & Revenue Analysis Dashboard
## Project Overview
This Power BI dashboard analyzes sales and revenue data using the Superstore dataset.

## Features
- Total Revenue KPI
- Total Orders KPI
- Sales by Region
- Sales by Segment
- Sales by Category
- Sales Trend Over Time
- Interactive Dashboard

## Tools Used
- Power BI
- CSV Dataset
- Data Visualization
